# MINISPRIN 官方网站开发规范

## 项目基本信息

- 项目名称：MINISPRIN 官方网站
- 游戏全称：关于我和我的mini小组穿越到异世界，被迫组成勇者小队打倒魔王这件事，或许搞错了什么？
- 版权归属：MUXI Studio
- 技术栈：纯 HTML + CSS + 原生 JavaScript，单页滚动，仅适配桌面端（1280px+）
- 文件结构：index.html / css/style.css / js/main.js / assets/images/

---

## 页面板块结构（顺序不可更改）

1. Hero 区域（Landing）—— 深色背景，全屏
2. 关于本作（About）
3. 世界观（world
4. 实机演示（Demo）—— 三栏交互布局
5. 开发日志（Changelog）
6. 开发团队（Team）—— 5名成员
7. 开始冒险（Play）
8. 页脚（Footer）

## 视觉风格

整体定位：70% 柔和简约高级科技感 + 30% 日式轻卡通

参考气质：明日方舟官网加载页（冷色调、大留白、细线条、轻盈几何感），
局部保留圆润卡片、柔和色块等轻卡通温度感。

### 背景策略
- Hero 区域：深色背景 #0D0F1E，配动态粒子效果（细白色粒子缓慢漂浮）
- 其余内容板块：浅色背景 #F0F4FF，通过色块、装饰线、卡片投影丰富层次
- 每个内容板块至少使用以下一种装饰手段：
  - 板块背景加半透明大号英文装饰字（字号 120px+，透明度 5-8%）
  - 标题左侧加 3px 主色竖线装饰
  - 卡片 hover 时出现彩色顶部边框
  - 板块间使用斜切分隔线（clip-path skew）


### 字体
- 英文标题：Bebas Neue 或 Rajdhani（Bold），letter-spacing: 0.15em
- 中文正文：Noto Sans SC，字重 300-400
- 正文字号：16px，行高 1.75
- 标题层级：H1 约 72px / H2 约 32px / H3 约 20px，字重不超过 500



### 装饰元素
- 图标统一使用 Lucide Icons（线性风格）
- 禁止使用像素字体、8-bit 装饰、像素边框等强像素元素
- 允许少量细腻 CSS 动画：fade-up 入场、按钮 hover 微缩放
- 背景可用极淡几何线条网格或径向渐变光晕（透明度低于 15%）

## 交互规范

- 页面滚动时各板块用 IntersectionObserver 触发 fade-up 入场动画
  （translateY 20px → 0，opacity 0 → 1，duration 600ms）
- 所有可点击元素必须有 hover 状态反馈
- 按钮 hover 时加柔和光晕（box-shadow 扩散动画）
- 导航栏滚动时当前板块高亮（发光下划线效果）
- 顶部导航栏：毛玻璃效果（backdrop-filter: blur，半透明背景）
- 平滑滚动：scroll-behavior: smooth
- 所有外链在新标签页打开
- 禁止添加任何自动播放声音的元素

## 各板块关键规格

### Hero 区域
- 全屏 100vh，内容垂直居中
- 粒子背景（canvas，细白色粒子，低密度慢速漂浮）
- 元素层次：大 Logo → 中文副标题（max-width 640px）→ 版本号 pill → 按钮组
- 主按钮「进入网站 ↓」强调色填充；次按钮「GitHub 仓库」白色描边

### 实机演示（三栏布局，高度固定 640px）
- 左侧导航栏（约 130px）：5个标签，顺序固定为「战斗」「强化」「抽卡」「图鉴」「成就」
  - 选中项：主色背景 + 白色文字 + 左侧竖线
  - 未选中项：hover 时浅色高亮
  - 切换时中间截图与右侧文字同步 fade 过渡（250ms）
- 中间截图区（约 45%）：竖屏手机外框 mockup（纯 CSS 绘制，圆角矩形细描边）
  - 截图比例严格 9:19.5（iPhone 14 Pro Max 竖屏）
  - 手动翻页：左右箭头「‹」「›」位于手机框两侧
  - 底部圆点指示器
  - 截图切换：水平滑动动画（CSS transform + transition）
  - 每个玩法预设 3 张占位截图（placehold.co，430×932）
- 右侧文字区（约 30%）：玩法名称 + 2-3 段介绍，切换时 fade 更新

### 开发团队
- 5名成员，5列等宽横向排列
- 职位构成：产品 × 1、前端 × 2、后端 × 1、设计 × 1
- 职位标签颜色：产品→蓝紫，前端→青绿，后端→橙，设计→粉
- 每张卡片：头像（圆形 80×80px）、姓名、职位标签、留言、联系方式
- 联系方式默认隐藏，点击邮件/GitHub 图标后 max-height 过渡展开
- 卡片 hover：translateY(-4px)，box-shadow 柔和加深

### 更新日志
- 竖向时间轴，版本倒序
- 连线：1px 细线，主色低透明度；节点：实心圆点（主色）
- 整体简洁，类技术文档风格

## 间距与圆角系统

- 间距系统：4px 的倍数（8 / 16 / 24 / 32 / 48 / 64 / 96px）
- 各板块上下 padding：100px
- 卡片圆角：12px
- 按钮圆角：8px
- 标签圆角：99px（pill）
- 所有板块标题（H2）统一居中对齐
